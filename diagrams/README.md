# diagrams/

This folder contains signal flow charts, system block diagrams, and schematic references created as part of the Capture-to-Reproduction Audio Study.

## Purpose

- Signal chain block diagrams illustrating the path from microphone to loudspeaker
- Crossover network schematics and topology comparisons
- Enclosure geometry sketches and port placement diagrams
- DSP processing flow charts (EQ, time alignment, room correction)
- Automotive cabin acoustic diagrams (speaker placement, boundary interactions)

## Status

No diagrams have been created yet. This folder will be populated as design and analysis work progresses.

## File Formats

Diagrams may be stored as vector graphics (SVG), portable documents (PDF), or raster images (PNG) depending on how they are produced. Source files (e.g., draw.io XML, KiCad schematics) will be included alongside exported images where applicable.
